# Matrix Multiplication Calculator

![Python](https://img.shields.io/badge/Python-3.6%2B-blue.svg)
![Mathematics](https://img.shields.io/badge/Mathematics-Matrix%20Operations-green.svg)

Программа для умножения матриц и сравнения результатов. Выполняет операцию `S * J * S1` и сравнивает полученный результат с матрицей `A`.

## Назначение

Программа предназначена для:
- Умножения трех матриц последовательно: `S × J × S1`
- Сравнения результата с заданной матрицей `A`
- Визуализации матриц в удобном формате
- Проверки совместимости размеров матриц

## Функциональность

### Основные возможности:
- **Ввод матриц** с клавиатуры с валидацией данных
- **Последовательное умножение** трех матриц
- **Проверка совместимости** размеров матриц
- **Красивый вывод** матриц рядом для сравнения
- **Обработка ошибок** ввода и вычислений

### Особенности:
- ✅ Валидация вводимых данных
- ✅ Проверка размеров матриц для умножения
- ✅ Форматированный вывод с выравниванием
- ✅ Подробные сообщения об ошибках
